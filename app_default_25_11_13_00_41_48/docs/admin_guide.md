# Przewodnik Administratora Systemu

## Zarządzanie użytkownikami i rolami

- Używaj API `/users` do tworzenia, edycji i usuwania użytkowników
- Zarządzaj rolami przez endpointy `/admin/roles`
- Przypisuj uprawnienia w kolumnie `permissions` roli
